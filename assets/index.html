    What is CryptoZ ?

    CryptoZ is a cryptography implementation web app that have many features and tools to encrypt or decrypt message. This website will also give you informations about how encryption and decryption for each tools work.

    Features

    * Classic Cryptography

    Encrypt and decrypt text using some classic cryptography technique.

    * Steganography

    Hide text to an image, or extract hidden message from an Image.
